### README:

*To run this program, please follow the following steps:*

1. open a terminal window and navigate to the folder where you unzipped the program files
2. run the command `make` in the terminal window
3. run the command `./recursion` in the terminal window
4. follow the on-screen instructions to choose a menu option
5. choose 0 to exit
6. run the command `make clean` in the terminal window to leave the directory in its original state
